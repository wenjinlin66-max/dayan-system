# 输入字段规范

## 必填字段

- `distance_cm`：发射与接收线圈相对距离（cm）
- `M_H`：Maxwell 导出的互感（H）

## 推荐字段

- `L1_H`：发射侧自感（H）
- `L2_H`：接收侧自感（H）

## 可选字段（用于三维曲面）

- `x_cm`：水平偏移（cm）
- `z_cm`：垂直偏移（cm）

## 计算参数（命令行给定）

- `--freq`：频率 Hz，默认 `85000`
- `--r1`：发射侧等效电阻，默认 `0.3`
- `--r2`：接收侧等效电阻，默认 `0.3`
- `--rload`：负载电阻，默认 `10`

## 说明

- 若缺少 `L1_H/L2_H`，脚本会跳过 `k` 计算。
- 若缺少 `x_cm/z_cm`，脚本会跳过三维曲面图。
- 输入单位错误会导致效率计算失真，尤其是 `M_H` 必须是 H 而不是 uH。

## 自动列名识别（新增）

- 你可以直接用 Maxwell 原始导出列，不需要手动改名。
- 脚本会自动识别常见别名（中英文混合），例如：
  - `M`, `mutual_inductance`, `互感` -> `M_H`
  - `L1`, `发射侧自感` -> `L1_H`
  - `L2`, `接收侧自感` -> `L2_H`
  - `distance_mm` -> 自动换算为 `distance_cm`
  - `m_uH`, `l1_uH`, `l2_uH` -> 自动换算为 H

## 图编号参数（新增）

- `--fig-chapter`：论文章号，默认 `3`
- `--fig-start`：起始图序号，默认 `1`
- 会额外生成：
  - `figure_captions.md`
  - `figure_captions.csv`
  - `fig_<章号>_<序号>_*.png`
