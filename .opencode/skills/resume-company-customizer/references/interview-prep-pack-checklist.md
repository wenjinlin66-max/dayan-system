# 面试资料包检查单（与简历同一 Word 交付）

用于确保一次正常面试所需资料完整，避免只给“问题列表”而缺少可执行准备内容。

## A. 必备结构

- [ ] 附录A：公司与岗位详细介绍
- [ ] 附录B：面试高频问题与应对手段
- [ ] 附录C：完整面试资料包

## B. 附录B 质量门槛

- [ ] 题量 >= 12
- [ ] 每题包含：面试官考察意图 / 答题思路 / 示例回答要点 / 风险点 / 应对手段 / 证据锚点
- [ ] 连续 3 题首句不重复
- [ ] 证据锚点覆盖至少 2 段不同经历

## C. 附录C 最低清单

- [ ] C1 自我介绍包（30秒 / 60秒 / 90秒）
- [ ] C2 项目证据包（至少 2 个）
- [ ] C3 岗位术语与业务理解速记
- [ ] C4 STAR 案例库（至少 6 题）
- [ ] C5 专业问题速答卡（至少 10 题）
- [ ] C6 反问面试官清单（至少 8 条）
- [ ] C7 面试当天检查单（材料/设备/时间/路线/应急）

## D. 交付前自检

- [ ] 内容可直接用于演练，不是抽象口号
- [ ] 与目标岗位强相关，不灌水
- [ ] 不虚构证据与数据
- [ ] 可在 30 分钟内用于快速复习
