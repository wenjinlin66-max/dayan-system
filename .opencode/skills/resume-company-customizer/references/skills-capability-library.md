# Skill 能力库（按任务表达）

## 使用方式

- 先判断岗位核心能力，再从下列能力中选择 1-4 条高相关项。
- 每条能力用“做了什么 + 怎么做 + 产出什么”表达。
- 禁止只写工具名，不写任务与结果。

## 通用能力项

### Agent + MCP 工作流

- 可构建“信息采集 -> 内容提炼 -> 文档生成 -> 复核交付”的自动化流程。
- 可将重复任务流程化，提升资料处理与文档交付效率。

### docx 文档能力

- 可基于模板批量生成/改写 Word 文档（简历、报告、说明书）。
- 可在保持版式不变的前提下进行结构化内容更新。

### pdf 能力

- 可完成 PDF 文本提取、信息整合、合并拆分与结构化整理。
- 可将 PDF 内容转为可编辑材料用于后续文档生产。

### pptx 能力

- 可根据主题快速生成演示文稿结构，完成内容改写与页面信息优化。
- 可按受众和场景定制汇报逻辑，提高表达清晰度。

### xlsx 能力

- 可进行表格清洗、字段重组、基础统计与结果可视化整理。
- 可将分散数据整合为可汇报的数据摘要。

## 岗位映射建议

### 机械/硬件岗位

- 写法重点：机械设计、仿真验证、工程落地。
- Skill 能力作为补充：强调技术文档生成、测试记录整理、跨团队资料协同。

### 软件/数据岗位

- 写法重点：自动化流程、工具链集成、稳定交付。
- Skill 能力可作为主体之一：强调 Agent+MCP 工作流搭建与效率提升。

### 运营/职能岗位

- 写法重点：信息处理、报告交付、流程规范。
- Skill 能力可写为核心执行力：文档、报表、演示材料一体化生产。

## 示例句子

- 基于 Agent + MCP 构建简历定制工作流，实现公司信息采集、岗位关键词提炼与 Word 定制化生成的闭环交付。
- 可使用 docx/pdf/pptx/xlsx 等 Skill 完成文档改写、资料整合、汇报材料制作与结构化输出，提升多场景交付效率。
