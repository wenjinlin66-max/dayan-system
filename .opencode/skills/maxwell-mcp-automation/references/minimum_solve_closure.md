# Setup1 可解最小闭环清单（Eddy Current）

在 `run_setup` 前，至少满足以下条件：

1. 几何体
- 发射线圈实体（或导体路径）
- 接收线圈实体（或导体路径）
- 空气域（包裹线圈）

2. 材料
- 线圈指定导电材料（如铜）
- 空气域指定空气材料

3. 激励与边界
- 给线圈施加激励（电流或电压，至少一侧）
- 设置空气域边界（开边界/辐射边界等）

4. 求解设置
- `Setup1` 已创建且频率参数与设计变量一致（建议引用 `f_op`）

5. 网格
- 默认网格可先跑通；建议在线圈附近加密网格以提升结果稳定性

6. 结果导出目标
- 至少定义需要导出的量：`L1`, `L2`, `M`, `k`

若任一项缺失，`Setup1` 可能返回求解失败或空结果。
